Seletores personalizados

Oque são os seletores personalizados?

Os seletores personalizados são um conjunto de indicadores que são responsáveis por personalizar e criar comandos com mais liberdade, sem que necessáriamentes esses comandos dados valham para todas as tags de mesmo nome, ou seja, são excessões. A seguir veremos mais sobre eles e oque cada um faz.

id

A tag id é um identificador, ela serve para ser usada em casos onde queremos fazer alterações para um item específico, sem que passe a valer para os demais, por exemplo, que apenas um dos vários subtitulos (h2) de um site tenha a cor azul, sem que os demais fiquem desta mesma cor também. Geralmente pode ser usado para diferenciar 1 item.

class

É um identificador que marca vários itens ao invés de 1, se eu quiser que todos os subtitulos (h2) tenham a cor azul, todos os eles terão com um só comando, e caso, eu queira alterar a cor futuramente, basta voltar ao comando e alterar a cor e será válido em todos os itens.

Pseudo classes

As pseudo classes estão ligadas ao estado do item, por exemplo, quando eu passar o mouse em cima do termo, ele irá mudar de cor, ou qual formarto, ou qual cor.

Pseudo elementos

Um pseudo-elemento CSS é uma palavra-chave adicionada a um seletor que permite que você estilize uma parte específica do elemento selecionado.
