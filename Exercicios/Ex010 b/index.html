Listas Ordenadas e não ordenadas - Ex 010b

Neste exercício veremos a respeito das listas ordenadas, não ordenadas e de definição.

Listas ordenadas

No exemplo abaixo estamos vendo uma lista manual e sem tag ou código de comando algum, suponhamos que na verdade você leitor cabeça de pika, por algum motivo que só Deus sabe, prefere estudar logo depois que acorda, sem nem escovar os dentes nem nada, já mete a cara no caderno com esse teu bafo de merda mesmo.

1 - acordar 2- estudar 2 - escovar os dentes 3 - comer algo 4 - estudar

A primeira coisa a se notar é que, o navegador não consegue reconhecer isso como uma lista, por isso é um recurso visual que você usou e não está no código, não tem significado algum e portanto está todo errado ná pagina, na mesma linha. O segundo problema é que, ao por a terefa ali acima temos duas tarefas com números iguais, você pode até me dizer que isso é besteira e que dá pra só corrigir da meneira correta mas quero ver você fazer isso com uma lista de 300 itens.

Neste caso usamos a tag "ol" (listas ordenadas), para abrir uma lista e usamos a tag "li" para os itens.

  1. acordar
  2. estudar
  3. escovar os dentes
  4. comer algo
  5. estudar

Utilizando o comando "type" você pode alterar o formato da lista e com o comando "start onde ela começa."

  1. acordar
  2. estudar
  3. escovar os dentes
  4. comer algo
  5. estudar

Aqui estamos usando a lista com a tag "ol" ativa e com inicio a partir do número 3.

  1. acordar
  2. estudar
  3. escovar os dentes
  4. comer algo
  5. estudar

Não é obriogatório por o "li" abrir a "ol" e por os itens é valido também.

Listas não ordenadas

Nas listas não ordenadas temos esse formatato diferente,not que ela não tem números nem letras, ela tem pontos. Podemos alter também o formato desses indicadores em "type" e selecionando as opções.

  • acordar
  • escovar os dentes
  • comer
  • estudar
  • arrumar a casa

com o "type" habilitado.

  • acordar
  • escovar os dentes
  • comer
  • estudar
  • arrumar a casa

Listas de definição

Listas de definição são o modelo que você utiliza quando quer fazer uma lista dentro de outra lista. para fazer isso insira a tag "ol" abaixo do nome da lista.

Minhas músicas favoritas

  1. Antigas
    1. Como tudo deve ser Charlie Brown Jr
    2. Cafuné (esqueci o nome)
    3. Céu azul
    4. Pontes indestrutiveis
    5. tanto faz
  2. Novas
    1. Caju
    2. Se eu usasse sapato
    3. espetaculo estranho
    4. Nas ruas
    5. Distante de mais

* Lembre-se que, podemos sempre ir abrindo listas dentro de listas.

Lista de definição

É uma lista para termo e descrição do item. Como um dicionário por exemplo.

feliz
estar contente, de bem com a vida.
triste
Não estar contente, de mal com a vida.

"dl" é a tag para criação de lista, "dt" é o termo e "dd" a descrição.
