Links internos

Neste exercício veremos como acessar sites utilizando links, vamos utilizar a tag "a" de ancôra. Como por exemplo o github que é o site do professor e também tem o canal do curso em vídeo

Neste caso ao abrir o link, o site se sobrepôs ao meu, para evitar qe isso aconteça bast inserir as tags "target="_bank rel="external". Vou por o mesmo site abaixo e ao clicar nele será aberta uma nova guia.

links internos

Os links externos vão basicamente referênciar uma pasta já existete do usuário, existem dois tipos de links internos

Está é a primeira página deste arquivo, caso queira saber mais informações clique aqui

* Apertando ctrl + space o code vai te mostrar as páginas disponíveis.

* O "rel" basicamente vai servir pra dar sentido a próxima página. Ou para indicar que o arquivo está duplicado, que há mais conteúdo na página. Existem variações de comandos em que o hel vai informar para o mecanismo de busca sobre a página, se por exemplo eu quiser por sobre o patrocinador desta página, o professor Guanabara com o seu site e você leitor quiser realmente saber mais sobre você vai clicar e vais er encaminhado normalmente, oque vai mudar e que com o comando "rel="nofollow" eu vou informar para o algoritimo que eu não tenho ligação com este site, ele não é meu, é um site de um patrocionador.

Você também pode acessar nossa página de notícias para ficar sabendo das principais informações do momento !
