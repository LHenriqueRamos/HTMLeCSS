tag de identação : "pre"

Neste caso estamos vendo o exemplo de um código fonte escrito dentro da tag "code", a princípio pode-se pensar que, por estar dentro de uma tag de comando, o código será representado da maneira correta, porém não é isso que acontece, como podemos ver o comando está formatado como texto.

Exemplo A

num= int (input('digite um número')) if num 2 % == 0: print(f 'O número {num} é PAR') else: print(f 'O número {num} é ÍMPAR') print('Fim do programa')

Exemplo B

Para corrigir este erro, utilizamos a tag "pre" para representa-lo como código

        num= int (input('digite um número'))
        if num 2 % == 0:
        print(f 'O número {num} é PAR') else:
        print(f 'O número {num} é ÍMPAR')
        print('Fim do programa')

Ao utilizar a tag correta podemos observar duas coisas, a primeira é que, graças a tag o texto foi devidamente formatado, desta forma o usuário que hipotéticamente ler, poderá ter uma melhor compreenção. O segundo ponto a ser observado é que, houve uma junção de comandos, a tag já utilizada junto a tag code mostrada anteriomente, que acabou por padronizar o formato dos caracteres dando-lhes um resultado mais homogênio.
