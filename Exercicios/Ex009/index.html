Tags adiconais

Texto marcado

Podemos utilizar a tag "mark" para fazer a marcação num documento. Não se usa alterar a cor no HTML, isso é feito no CSS mas existem alguns parâmetros para isso como é o caso de por a tag "style" dentro da tag que cê já fez, Exemplo:

"mark style = line texto de exemplo /mark"

Como já foi dito anteriomente, essa é uma tag de estilo, formato, não semântica então se é o caso de querer alterar cor ou coisas parecida, faça no CSS, é o jeito certo de fazer a coisa.
